Char-Level Counting: Where Should Stats Live?

Three approaches for surfacing live text metrics while writing

A

Status Strip (Recommended)

A slim, always-visible bar between the textarea and bottom toolbar. Shows: current sentence chars · paragraph chars/words · total chars/words/sentences · reading time. Compact single line. Click any segment to expand details. Selection count replaces sentence count when text is selected.

S: 42c 8w    ¶: 187c 31w 4s    ∑: 1,247c 203w 18s · 1m read
B

Panel Integration

Add a "Text Stats" section to the existing right-side syntax panel. Shows the same metrics but in a vertical card layout. Always visible on desktop, accessible via the harmonica gesture on mobile. Less intrusive but requires opening the panel.

C

Hybrid: Strip + Panel

Compact strip shows sentence + total counts (always visible). Detailed breakdown lives in the panel. Best of both worlds but more UI surface area. The strip is the "glanceable" layer, the panel is the "deep dive" layer.
